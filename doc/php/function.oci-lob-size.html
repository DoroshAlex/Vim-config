OCI-Lob->size

(PHP 5, PECL OCI8 >= 1.1.0)

OCI-Lob->size — Возвращает размер объекта LOB

Описание

int OCI-Lob::size ( void )

Возвращает размер объекта LOB

Возвращаемые значения

Возвращает длину объекта LOB или FALSE в случае возникновения ошибки. Для пустых объектов длина равняется нулю.
