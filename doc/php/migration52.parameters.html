Новые параметры

В PHP 5.2.x у некоторых функций появились новые, необязательные параметры:

Ядро PHP:

  • htmlentities() - добавлен параметр double_encode в PHP 5.2.3.
  • htmlspecialchars() - добавлен параметр double_encode в PHP 5.2.3.
  • base64_decode() - добавлен параметр strict
  • setcookie() - добавлен параметр httponly
  • setrawcookie() - добавлен параметр httponly
  • session_set_cookie_params() - добавлен параметр httponly
  • memory_get_usage() - добавлен параметр real_usage
  • get_loaded_extensions() - добавлен параметр zend_extensions в PHP 5.2.4

curl:

  • curl_multi_info_read() - добавлен параметр msgs_in_queue

datetime

  • date() - добавлен символ форматирования "u" (миллисекунды) в PHP 5.2.2

imap:

  • imap_open() - добавлен параметр n_retries
  • imap_reopen() - добавлен параметр n_retries

mbstring:

  • mb_strrpos() - добавлен параметр offset
    Внимание

    Параметр offset добавлен на место параметра encoding. Обратная совместимость обеспечивается указанием encoding как третьего параметра. Однако использовать это не рекомендуется, так как поддержка encoding как третьего параметра будет удалена в последующих версиях PHP.

ming:

  • swfmovie::streamMP3() - добавлен параметр skip в PHP 5.2.1

openssl:

  • openssl_verify() - добавлен параметр signature_algo

pgsql:

  • pg_escape_bytea() - добавлен параметр connection
  • pg_escape_string() - добавлен параметр connection

simplexml:

  • SimpleXMLElement::__construct() - добавлен параметр is_prefix
  • SimpleXMLElement::attributes() - добавлен параметр is_prefix
  • SimpleXMLElement::children() - добавлен параметр is_prefix
  • simplexml_load_file() - добавлен параметр is_prefix
  • simplexml_load_string() - добавлен параметр is_prefix

spl:

  • array iterator_to_array(Traversable it [, bool use_keys = true]) - добавлен параметр use_keys в PHP 5.2.1

xmlreader:

  • XMLReader::open() - добавлены параметры encoding и options
  • XMLReader::XML() - добавлены параметры encoding и options

XMLWriter:

  • XMLWriter::WriteElement() - параметр content стал необязательным в PHP 5.2.3
  • XMLWriter::WriteElementNs() - параметр content стал необязательным в PHP 5.2.3
