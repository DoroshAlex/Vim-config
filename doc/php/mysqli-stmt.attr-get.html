mysqli_stmt::attr_get

mysqli_stmt_attr_get

(PHP 5)

mysqli_stmt::attr_get -- mysqli_stmt_attr_get — Получает текущее значение атрибута запроса

Описание

Объектно-ориентированный стиль

int mysqli_stmt::attr_get ( int $attr )

Процедурный стиль

int mysqli_stmt_attr_get ( mysqli_stmt $stmt , int $attr )

Используется для получения текущего значения атрибута запроса.

Список параметров

stmt

Только для процедурного стиля: Идентификатор выражения, полученный с помощью mysqli_stmt_init().

attr

Атрибут, который необходимо получить.

Возвращаемые значения

Возвращает FALSE если атрибут не найден, либо возвращает текущее значение атрибута.
