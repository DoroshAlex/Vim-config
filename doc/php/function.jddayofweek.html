JDDayOfWeek

(PHP 4, PHP 5)

JDDayOfWeek — Возвращает день недели

Описание

mixed jddayofweek ( int $julianday [, int $mode = CAL_DOW_DAYNO ] )

Возвращает день недели. Может возвращать строку или число в зависимости от режима работы.

Список параметров

julianday

День в виде числа в Юлианском летоисчислении

mode
Режимы отображения дней недели
Режим Значение
0 (По умолчанию) День недели возвращается в виде числа (0=Воскресенье, 1=Понедельник и т.д.)
1 Возвращается строка содержащая день недели (Английский-Григорианский)
2 Возвращается строка содержащая аббревиатуру дня недели (Английскую-Григорианскую)

Возвращаемые значения

День недели по григорианскому календарю в виде строки или числа.
