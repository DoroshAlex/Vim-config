AMQPConnection::setHost — Установить хост AMPQ

Описание

public bool AMQPConnection::setHost ( string $host )

Этот метод устанавливает имя хоста, которое будет использоваться при соединении с брокером AMQP.

Список параметров

host

Имя хоста брокера AMQP.

Возвращаемые значения

Возвращает TRUE в случае успешного завершения или FALSE в случае возникновения ошибки.

Ошибки

Выдает исключение AMQPConnectionException, если host длиннее 1024 символов.

Примеры

Пример #1 Пример использования AMQPConnection::setHost()

<?php

/* Создание нового соединения */
$cnn = new AMQPConnection();

// установка имени хоста
$cnn->setHost('myhost.com');

?>
