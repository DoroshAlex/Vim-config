OCI-Lob->append

(PHP 5, PECL OCI8 >= 1.1.0)

OCI-Lob->append — Добавляет данные из объекта LOB в конец другого объекта

Описание

bool OCI-Lob::append ( OCI-Lob $lob_from )

Добавляет в конец объекта LOB данные из другого объекта LOB.

Запись в LOB с помощью OCI-Lob->append() вызовет ошибку, если у объекта включен режим буферизации. Поэтому вы должны выключить буферизацию перед применением. Перед выключением буферизации вам необходимо использовать OCI-Lob->flush для очистки буфера.

Список параметров

lob_from

Копируемый LOB.

Возвращаемые значения

Возвращает TRUE в случае успешного завершения или FALSE в случае возникновения ошибки.
