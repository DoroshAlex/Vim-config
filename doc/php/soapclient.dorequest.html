SoapClient::__doRequest

(PHP 5 >= 5.0.1)

SoapClient::__doRequest — Выполняет SOAP-запрос

Описание

public string SoapClient::__doRequest ( string $request , string $location , string $action , int $version [, int $one_way = 0 ] )

Выполняет SOAP-запрос поверх HTTP.

Этот метод может быть переопределен в подклассах для реализации других транспортных уровней, выполнения дополнительной обработки XML или других целей.

Список параметров

request

XML SOAP запрос.

location

URL для запроса.

action

SOAP-действие.

version

Версия SOAP.

one_way

Если one_way равен 1, то метод ничего не возвращает. Этот параметр используется, когда не ожидается ответа.

Возвращаемые значения

XML SOAP ответ.

Список изменений

Версия Описание
5.1.3 Добавлен параметр one_way.

Примеры

Пример #1 Пример использования SoapClient::__doRequest()

<?php
function Add($x,$y) {
  return $x+$y;
}

class LocalSoapClient extends SoapClient {

  function __construct($wsdl, $options) {
    parent::__construct($wsdl, $options);
    $this->server = new SoapServer($wsdl, $options);
    $this->server->addFunction('Add');
  }

  function __doRequest($request, $location, $action, $version, $one_way = 0) {
    ob_start();
    $this->server->handle($request);
    $response = ob_get_contents();
    ob_end_clean();
    return $response;
  }

}

$x = new LocalSoapClient(NULL,array('location'=>'test://', 
                                   'uri'=>'http://testuri.org')); 
var_dump($x->Add(3,4));
?>
