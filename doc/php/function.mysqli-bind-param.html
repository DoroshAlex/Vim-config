mysqli_bind_param

(PHP 5)

mysqli_bind_param — Синоним функции mysqli_stmt_bind_param()

Описание

Эта функция является синонимом функции mysqli_stmt_bind_param().

Примечания

Замечание:

Функция mysqli_bind_param() не представляет ценности и будет удалена.

  • mysqli_stmt_bind_param() - Binds variables to a prepared statement as parameters
