DomDocument->create_element_ns

(PHP 4 >= 4.1.0)

DomDocument->create_element_ns — Создает новый элемент с соответствующим пространством имен

Описание

domelement DomDocument->create_element_ns ( string $uri , string $name [, string $prefix ] )

Эта функция создает новый экземпляр класса DomElement. Имя тэга элемента берется из аргумента name. URI пространства имен берется из аргумента uri. Если пространство имен с таким же URI уже объявлено в корневом узле документа, префикс будет взят оттуда, в противном случае префикс будет взят из необязательного аргумента prefix, либо сгенерирован случайным образом. Этот узел не будет отображаться в документе до тех пор, пока он не будет вставлен, например, функцией DOMNode->appendChild().

В случае ошибки метод вернет FALSE.

См. также domdocument_create_element_ns(), domnode_add_namespace(), domnode_set_namespace(), domnode_append_child(), domdocument_create_text(), domdocument_create_comment(), domdocument_create_attribute(), domdocument_create_processing_instruction(), domdocument_create_entity_reference() и domnode_insert_before().
