sinh

(PHP 4 >= 4.1.0, PHP 5)

sinh — Гиперболический синус

Описание

float sinh ( float $arg )

Возвращает гиперболический синус arg, определяемый как (exp(arg) - exp(-arg))/2.

Список параметров

arg

Входное значение

Возвращаемые значения

Гиперболический синус arg
