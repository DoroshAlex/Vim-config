Настройка во время выполнения

Поведение этих функций зависит от установок в php.ini.

Опции настройки Iconv
Имя По умолчанию Меняемо Список изменений
iconv.input_encoding "ISO-8859-1" PHP_INI_ALL Поддерживается начиная с PHP 4.0.5.
iconv.output_encoding "ISO-8859-1" PHP_INI_ALL Поддерживается начиная с PHP 4.0.5.
iconv.internal_encoding "ISO-8859-1" PHP_INI_ALL Поддерживается начиная с PHP 4.0.5.

Краткое разъяснение конфигурационных директив.

Внимание

Некоторые системы (например IBM AIX) используют "ISO8859-1" вместо "ISO-8859-1", поэтому это значение должно быть указано в настройках и параметрах функции.

iconv.input_encoding string

В настоящее время эта опция не используется.

iconv.output_encoding string

iconv.internal_encoding string
