DomNode->replace_node

(PHP 4 >= 4.2.0)

DomNode->replace_node — Заменяет узел

Описание

domelement DomNode->replace_node ( domelement $newnode )

(PHP 4.2) Эта функция заменяет существующий узел на новый переданный в качестве аргумента. Перед заменой newnode копируется, если у него есть родитель, чтобы быть уверенными, что уже существующий узел не будет вставлен повторно. Такое поведение функции вынуждает делать все изменения в узле до замены, либо выбирать уже добавленный узел после замены такими функциями, как domnode_first_child(), domnode_child_nodes() и т.п.

(PHP 4.3) Эта функция заменяет существующий узел на новый переданный в качестве аргумента. Если newnode уже был добавлен в документ, он будет отсоединен от своего прежнего контекста. В случае успешной замены функция вернет старый узел.

См. также domnode_append_child()
