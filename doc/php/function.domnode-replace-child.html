DomNode->replace_child

(PHP 4 >= 4.2.0)

DomNode->replace_child — Замена сына

Описание

domelement DomNode->replace_child ( domelement $newnode , domelement $oldnode )

(PHP 4.2) Эта функция заменяет сына oldnode новым переданным функции узлом. Если новый узел уже является сыном, то второе добавление не производится. Если старый узел найти не удается, функция вернет FALSE. Если замена прошла успешна, функция возвращает старый узел.

(PHP 4.3) Эта функция заменяет сына oldnode новым переданным функции узлом newnode, даже если новый узел уже является сыном DomNode. Если newnode уже был вставленным в документ, он отсоединяется от своего предыдущего контекста. Если заменяемый узел не удалось найти, функция вернет FALSE. Если замена завершилась успешно, будет возвращен старый узел. (Это поведение отвечает требованиям W3C спецификации.)

См. также domnode_append_child()
