octdec

(PHP 4, PHP 5)

octdec — Переводит число из восьмеричной системы счисления в десятичную

Описание

number octdec ( string $octal_string )

Возвращает десятичный эквивалент восьмеричного числа, указанного в аргументе octal_string.

Список параметров

octal_string

Восьмеричная строка для преобразования

Возвращаемые значения

Десятичное представление octal_string

Список изменений

Версия Описание
С 4.1.0 Теперь функция может преобразовывать числа, слишком большие для того, чтобы вместиться в тип integer для имеющейся платформы. В этом случае значения возвращаются как тип float.

Примеры

Пример #1 Пример использования octdec()

<?php
echo octdec('77') . "\n";
echo octdec(decoct(45));
?>

Результат выполнения данного примера:

63
45

Смотрите также

  • decoct() - Переводит число из десятичной системы счисления в восьмеричную
  • bindec() - Двоичное в десятичное
  • hexdec() - Переводит число из шестнадцатиричной системы счисления в десятичную
  • base_convert() - Преобразование числа между произвольными системами счисления
