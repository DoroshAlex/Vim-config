JsonSerializable::jsonSerialize

(PHP 5 >= 5.4.0)

JsonSerializable::jsonSerialize — Уточняет данные, которые должны быть сериализованы в JSON

Описание

abstract public mixed JsonSerializable::jsonSerialize ( void )

Внимание

К настоящему времени эта функция еще не была документирована; для ознакомления доступен только список аргументов.

Список параметров

У этой функции нет параметров.

Возвращаемые значения

Возвращает данные, которые должны быть сериализованы json_encode().
