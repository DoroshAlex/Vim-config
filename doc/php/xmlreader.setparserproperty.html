XMLReader::setParserProperty

(PHP 5 >= 5.1.2)

XMLReader::setParserProperty — Устанавливает опцию парсера

Описание

bool XMLReader::setParserProperty ( int $property , bool $value )

Устанавливает опции для парсера. Опции должны быть установлены после вызова XMLReader::open() или XMLReader::xml() и до первого вызова XMLReader::read().

Список параметров

property

Одна из констант опций парсера.

value

Если установлено в TRUE, то опция будет включена, в противном случае опция будет выключена.

Возвращаемые значения

Возвращает TRUE в случае успешного завершения или FALSE в случае возникновения ошибки.
