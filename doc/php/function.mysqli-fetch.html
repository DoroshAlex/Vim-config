mysqli_fetch

(PHP 5)

mysqli_fetch — Синоним для mysqli_stmt_fetch()

Описание

Функция является синонимом mysqli_stmt_fetch().

Примечания

Замечание:

Функция mysqli_fetch() устарела и будет удалена.
