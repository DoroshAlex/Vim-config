AMQPConnection::isConnected

(PECL amqp >= Unknown)

AMQPConnection::isConnected — Определить, подключен ли объект AMQPConnection к брокеру

Описание

public bool AMQPConnection::isConnected ( void )

Этот метод проверяет является ли подключение к брокеру AMQP действительным. Это делается путем проверки возвращённого статуса последней команды.

Список параметров

У этой функции нет параметров.

Возвращаемые значения

Возвращает true, если соединение установлено, в противном случае false

Примеры

Пример #1 Пример использования AMQPConnection::isConnected()

<?php

/* Создание нового соединения */
$conn = new AMQPConnection();
$conn->connect();

/* Проверка работоспособности соединения */
if ($conn->isConnected()) {
    echo "Соединение с брокером установлено \o/";
}
else {
    echo "Невозможно подключиться к брокеру";
}

?>
