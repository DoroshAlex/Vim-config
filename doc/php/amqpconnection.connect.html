AMQPConnection::connect — Установить соединение с брокером AMQP

Описание

public bool AMQPConnection::connect ( void )

Этот метод инициирует соединение с брокером AMQP.

Список параметров

У этой функции нет параметров.

Возвращаемые значения

Возвращает TRUE в случае успешного завершения или FALSE в случае возникновения ошибки.

Примеры

Пример #1 Пример использования AMQPConnection::connect()

<?php

// Создание нового соединения
$cnn = new AMQPConnection();

// Установка логина и пароля пользователя
$cnn->setLogin('mylogin');
$cnn->setPassword('mypass');

if ($cnn->connect()) {
    echo "Соединение с брокером установлено";
}
else {
    echo "Невозможно установить соединение с брокером";
}


?>
