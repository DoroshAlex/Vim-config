printer_open

(PECL printer SVN)

printer_open — Открывает соединение с принтером

Описание

resource printer_open ([ string $printername ] )

Функция пытается открыть соединение с указанным принетром.

printer_open() также создает контекст устройства.

Список параметров

printername

Имя принтера. Если параметр не указан, функция пытается установить соединение с принтером по-умолчанию (если он не указан в php.ini как printer.default_printer, PHP попытается найти его самостоятельно).

Возвращаемые значения

В случае успеха возвращает дескриптор принтера или FALSE в случае возникновения ошибки.

Примеры

Пример #1 Пример printer_open()

<?php
$handle = printer_open("HP Deskjet 930c");
$handle = printer_open();
?>
