DomNode->node_name

(PHP 4 >= 4.1.0)

DomNode->node_name — Возвращает имя узла

Описание

string DomNode->node_name ( void )

Возвращает имя узла. Имя может отличаться для различных типов узлов, как показано в следующей таблице.

Значение имени
Тип Значение
DomAttribute значение атрибута
DomAttribute  
DomCDataSection #cdata-раздел
DomComment #comment
DomDocument #document
DomDocumentType имя типа документа
DomElement значение тэга имени элемента
DomEntity имя сущности
DomEntityReference имя ссылки на сущность
DomNotation имя обозначения
DomProcessingInstruction цель
DomText #text
