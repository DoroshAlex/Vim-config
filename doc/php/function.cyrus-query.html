cyrus_query

(PHP 4 >= 4.1.0, PECL cyrus 1.0)

cyrus_query — Послать запрос Cyrus IMAP серверу

Описание

array cyrus_query ( resource $connection , string $query )

Посылает запрос Cyrus IMAP серверу.

Список параметров

connection

Дескриптор соединения.

query

Строка запроса.

Возвращаемые значения

Возвращает ассоциативный массив со следующими ключами: text, msgno и keyword.
