OCI-Lob->write

(PHP 5, PECL OCI8 >= 1.1.0)

OCI-Lob->write — Записывает данные в объект LOB

Описание

int OCI-Lob::write ( string $data [, int $length ] )

Записывает данные из параметра data в текущую позицию объекта.

Список параметров

data

Записываемые данные.

length

Если указан параметр length, запись остановится после того, как length байт будет записано или закончатся данные из параметра data.

Возвращаемые значения

Возвращает количество записанных байт или FALSE в случае возникновения ошибки.
