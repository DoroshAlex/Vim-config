domxml_xslt_stylesheet

(PHP 4 >= 4.2.0)

domxml_xslt_stylesheet — Создает объект DomXsltStylesheet из строки XSL

Описание

DomXsltStylesheet domxml_xslt_stylesheet ( string $xsl_buf )

Создает объект класса DomXsltStylesheet из переданного XSL буфера.

Список параметров

xsl_buf

XSL документ в виде строки.

Возвращаемые значения

Возвращает новый экземпляр класса DomXsltStylesheet.

Переход на PHP 5

Вызывайте XSLTProcessor::importStylesheet() с аргументом DOMDocument::loadXML($xsl_buf).

Смотрите также

  • DomXsltStylesheet->process

  • domxml_xslt_stylesheet_doc() - Создает объект класса DomXsltStylesheet из объекта DomDocument
