domxml_xmltree

(PHP 4 >= 4.2.0)

domxml_xmltree — Создает дерево объектов PHP из XML документа

Описание

DomDocument domxml_xmltree ( string $str )

Функция разбирает XML документ из строки str и возвращает соответствующее дерево объектов PHP.

Эта функция изолирована от остальных, это означает, что невозможно получить доступ к дереву, используя другие функции. Изменение дерева, например добавление к нему узлов, не имеет смысла, потому как на данный момент нет способа получить из этого дерева обратно XML файл.

Тем не менее, эта функция может оказаться полезной, если требуется только прочитать файл и исследовать содержимое.

Список параметров

str

Содержимое XML файла.

Возвращаемые значения

Возвращает дерево DOM объектов, начиная с DomDocument.
