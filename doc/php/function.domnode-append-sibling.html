DomNode->append_sibling

(PHP 4 >= 4.2.0)

DomNode->append_sibling — Добавляет к узлу нового брата

Описание

domelement DomNode->append_sibling ( domelement $newnode )

Эта функция добавляет брата к существующему узлу. Сын может быть создан функциями domdocument_create_element(), domdocument_create_text() и т.п. или просто используя другой узел.

Прежде чем новый брат будет добавлен, он дублируется. Таким образом новый сын является полной копией, которую можно модифицировать, не изменяя состояния переданного в функцию узла. Если переданный узел имеет сыновей, они все также будут дублированы, это позволяет довольно просто дублировать большие части XML документа. Функция возвращает добавленного брата. Если планируется дальнейшая модификация добавленного узла, необходимо пользоваться возвращаемым значением.

Эта функция добавлена с целью предоставить поведение функции domnode_append_child(), как она работала до версии PHP 4.2.

См. также domnode_append_before().
