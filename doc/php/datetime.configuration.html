Настройка во время выполнения

Поведение этих функций зависит от установок в php.ini.

Настройки конфигурации даты/времени
Название По умолчанию Место изменения История изменений
date.default_latitude "31.7667" PHP_INI_ALL Доступно с версии PHP 5.0.0.
date.default_longitude "35.2333" PHP_INI_ALL Доступно с версии PHP 5.0.0.
date.sunrise_zenith "90.583333" PHP_INI_ALL Доступно с версии PHP 5.0.0.
date.sunset_zenith "90.583333" PHP_INI_ALL Доступно с версии PHP 5.0.0.
date.timezone "" PHP_INI_ALL Доступно с версии PHP 5.1.0.
Для подробного описания констант PHP_INI_*, обратитесь к разделу Где могут быть установлены параметры конфигурации.

Краткое разъяснение конфигурационных директив.

date.default_latitude float

Широта по умолчанию.

date.default_longitude float

Долгота по умолчанию.

date.sunrise_zenith float

Угол, под которым солнце светит во время восхода.

date.sunset_zenith float

Угол, под которым солнце светит во время заката.

date.timezone string

Временная зона, используемая по умолчанию всеми функциями даты/времени. До версии PHP 5.4.0 эта директива работала только при неустановленной переменной TZ. Порядок приоритета используемых временных зон описан на странице date_default_timezone_get(). Смотрите также Список поддерживаемых временных зон.

Замечание: Первые четыре опции настройки в настоящее время используются только в функциях date_sunrise() и date_sunset().
