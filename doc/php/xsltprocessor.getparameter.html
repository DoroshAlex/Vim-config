XSLTProcessor::getParameter

(PHP 5)

XSLTProcessor::getParameter — Возвращает значение параметра

Описание

string XSLTProcessor::getParameter ( string $namespaceURI , string $localName )

Возвращает параметр, если он раньше установлен с помощью XSLTProcessor::setParameter().

Список параметров

namespaceURI

Пространство имён URI параметра XSLT.

localName

Локальное имя параметра XSLT.

Возвращаемые значения

Значение параметра, или NULL, если оно не установлено.

Смотрите также

  • XSLTProcessor::setParameter() - Устанавливает значение параметра
  • XSLTProcessor::removeParameter() - Удаляет параметр
