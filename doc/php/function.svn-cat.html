svn_cat

(PECL svn >= 0.1.0)

svn_cat — Возвращает содержимое файла в репозитарии

Описание

string svn_cat ( string $repos_url [, int $revision_no ] )

Возвращает содержимое по URL repos_url файла в репозитарии по состоянию на ревизию номер revision_no (необязательный параметр).

Список параметров

repos_url

Путь URL к элементу в репозитарии.

revision_no

Номер ревизии (целое число) соответствующего элемента, по умолчанию - HEAD (последняя ревизия).

Возвращаемые значения

Возвращает содержимое элемента в репозитарии в случае успешного завершения или FALSE в случае возникновения ошибки.

Примечания

Внимание

Эта функция является ЭКСПЕРИМЕНТАЛЬНОЙ. Поведение этой функции, ее имя и относящаяся к ней документация могут измениться в последующих версиях PHP без уведомления. Используйте эту функцию на свой страх и риск.

Примеры

Пример #1 Простой пример использования функции svn_cat()

Этот пример показывает как получить содержимое файла ревизии 28:

<?php
$contents = svn_cat('http://www.example.com/svnroot/calc/gui.c', 28)
?>
