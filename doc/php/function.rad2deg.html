rad2deg

(PHP 4, PHP 5)

rad2deg — Преобразует значение из радианов в градусы

Описание

float rad2deg ( float $number )

Функция преобразует значение number из радианов в градусы.

Список параметров

number

Значение в радианах

Возвращаемые значения

Эквивалент number в градусах

Примеры

Пример #1 Пример использования rad2deg()

<?php

echo rad2deg(M_PI_4); // 45

?>
