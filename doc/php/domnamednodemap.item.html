DOMNamedNodeMap::item

(PHP 5)

DOMNamedNodeMap::item — Извлекает узел с заданным индексом

Описание

DOMNode DOMNamedNodeMap::item ( int $index )

Извлекает узел с заданным индексом index из объекта класса DOMNamedNodeMap.

Список параметров

index

Индекс узла в отображении.

Возвращаемые значения

Узел с индексом index в отображении или NULL, если индекс имеет недопустимое значение (больше или равен количеству узлов в отображении).
