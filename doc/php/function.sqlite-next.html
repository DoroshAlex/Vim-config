sqlite_next

SQLiteResult->next

SQLiteUnbuffered->next

(PHP 5, PECL sqlite >= 1.0.0)

sqlite_next -- SQLiteResult->next -- SQLiteUnbuffered->next — Переходит на следующую запись

Описание

bool sqlite_next ( resource $result )

Объектно-ориентированный стиль (метод):

bool SQLiteResult::next ( void )
bool SQLiteUnbuffered::next ( void )

sqlite_next() перемещает внутренний указатель результата запроса result на следующую запись.

Список параметров

result

Ресурс результата SQLite. Этот параметр не требуется при использовании объектно-ориентированного метода.

Возвращаемые значения

Возвращает TRUE в случае успеха, или FALSE, если записей больше нет.

Смотрите также

  • sqlite_seek() - Переходит к определенному номеру записи в буферизованном результирующем наборе
  • sqlite_current() - Возвращает текущую запись из результирующего набора в виде массива
  • sqlite_rewind() - Переходит к первой записи
