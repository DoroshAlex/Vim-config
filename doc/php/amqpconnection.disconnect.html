AMQPConnection::disconnect — Закрыть соединение с брокером AMQP

Описание

public bool AMQPConnection::disconnect ( void )

Этот метод закрывает соединение с брокером AMQP.

Список параметров

void

Возвращаемые значения

Возвращает TRUE, если соединение было успешно закрыто, иначе - FALSE.

Примеры

Пример #1 Пример использования AMQPConnection::disconnect()

<?php

/* Создание нового соединения */
$cnn = new AMQPConnection();

// установка логина и пароля пользователя
$cnn->setLogin('mylogin');
$cnn->setPassword('mypass');

if (!$cnn->connect()) {
 throw new Exception('Невозможно соединиться с брокером');
}

// Здесь можно делать что-то интересное

if (!$cnn->disconnect()) {
 throw new Exception('Невозможно отсоединиться от брокера');
}


?>
