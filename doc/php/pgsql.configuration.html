Настройка во время выполнения

Поведение этих функций зависит от установок в php.ini.

Параметры конфигурации PostgreSQL
Имя Значение по умолчанию Возможные значения Changelog
pgsql.allow_persistent "1" PHP_INI_SYSTEM  
pgsql.max_persistent "-1" PHP_INI_SYSTEM  
pgsql.max_links "-1" PHP_INI_SYSTEM  
pgsql.auto_reset_persistent "0" PHP_INI_SYSTEM Доступно с версии PHP 4.2.0.
pgsql.ignore_notice "0" PHP_INI_ALL Доступна с версии PHP 4.3.0.
pgsql.log_notice "0" PHP_INI_ALL Доступно с версии PHP 4.3.0.
Для подробного описания констант PHP_INI_*, обратитесь к разделу Где могут быть установлены параметры конфигурации.

Краткое разъяснение конфигурационных директив.

pgsql.allow_persistent boolean

Следует ли разрешить постоянное соединение с Postgres.

pgsql.max_persistent integer

Максимальное количество постоянных соединений на один процесс.

pgsql.max_links integer

Максимальное количество соединений Postgres на один процесс, включая постоянные соединения.

pgsql.auto_reset_persistent integer

Определять разорванные постоянные ссылки с pg_pconnect(). Требуются небольшие дополнительные ресурсы.

pgsql.ignore_notice integer

Игнорировать или нет уведомления от PostgreSQL.

pgsql.log_notice integer

Записывать ли в лог уведомления от PostgreSQL. Для записи в лог уведомлений также необходимо отключить директиву PHP pgsql.ignore_notice.
