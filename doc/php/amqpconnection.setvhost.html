AMQPConnection::setVhost — Установить виртуальный хост AMQP

Описание

public bool AMQPConnection::setVhost ( string $vhost )

Этот метод устанавливает имя виртуального хоста брокера AMQP, к которому будет производится соединение.

Список параметров

host

Виртуальный хост, который будет использоваться брокером AMQP.

Возвращаемые значения

Возвращает TRUE в случае успешного завершения или FALSE в случае возникновения ошибки.

Ошибки

Выдает исключение AMQPConnectionException, если host длинее 32 символов.

Примеры

Пример #1 Пример использования AMQPConnection::setVhost()

<?php

/* Создание нового соединения */
$cnn = new AMQPConnection();

// Установка хоста
$cnn->setVhost('myvhost');

?>
