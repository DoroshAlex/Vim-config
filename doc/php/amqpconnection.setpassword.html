AMQPConnection::setPassword — Установить пароль

Описание

public bool AMQPConnection::setPassword ( string $password )

Этот метод устанавливает пароль, которое будет использоваться при соединении с брокером AMQP.

Список параметров

password

Пароль, который будет использоваться при аутентификации брокером AMQP.

Возвращаемые значения

Возвращает TRUE в случае успешного завершения или FALSE в случае возникновения ошибки.

Ошибки

Выдает исключение AMQPConnectionException, если password длинее 32 символов.

Примеры

Пример #1 Пример использования AMQPConnection::setPassword()

<?php

/* Создание нового соединения */
$cnn = new AMQPConnection();

// Установка пароля
$cnn->setPassword('mypassword');

?>
