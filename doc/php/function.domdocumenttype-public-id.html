DomDocumentType->public_id

(PHP 4 >= 4.1.0)

DomDocumentType->public_id — Возвращает общедоступный (public) идентификатор типа документа

Описание

string DomDocumentType::public_id ( void )

Эта функция возвращает public-идентификатор типа документа.

Возвращаемые значения

Возвращает общедоступный идентификатор объекта класса DomDocumentType в виде строки.

Примеры

Следующий пример не выведет ничего.

Пример #1 Извлечение public id

<?php
include("example.inc");

if (!$dom = domxml_open_mem($xmlstr)) {
  echo "Ошибка при разборе документа\n";
  exit;
}

$doctype = $dom->doctype();
echo $doctype->public_id();
?>

Переход на PHP 5

Используйте свойство publicId объекта DOMDocumentType.
