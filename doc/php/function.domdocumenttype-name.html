DomDocumentType->name

(PHP 4 >= 4.1.0)

DomDocumentType->name — Возвращает имя типа документа

Описание

string DomDocumentType::name ( void )

Эта функция возвращает имя типа документа.

Возвращаемые значения

Возвращает имя объекта DomDocumentType в виде строки.

Примеры

Пример #1 Получение имени типа документа

<?php

include("example.inc");

if (!$dom = domxml_open_mem($xmlstr)) {
    echo "Ошибка при разборе файла\n";
    exit;
}

$doctype = $dom->doctype();
echo $doctype->name(); // chapter

?>

Используйте свойство name объекта DOMDocumentType.
