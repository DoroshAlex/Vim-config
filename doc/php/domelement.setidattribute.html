DOMElement::setIdAttribute

(PHP 5)

DOMElement::setIdAttribute — Объявляет атрибут с заданным именем ключевым атрибутом

Описание

void DOMElement::setIdAttribute ( string $name , bool $isId )

Объявляет атрибут name ключевым.

Список параметров

name

Имя атрибута.

isId

Установите значение в TRUE если требуется сделать name идентифицирующим, в противном случае ставьте FALSE.

Возвращаемые значения

Эта функция не возвращает значения после выполнения.

Ошибки

DOM_NO_MODIFICATION_ALLOWED_ERR

Возникает, если узел доступен только для чтения.

DOM_NOT_FOUND

Возникает, если name не является атрибутом элемента.

Смотрите также

  • DOMDocument::getElementById() - Ищет элемент с заданным id
  • DOMElement::setIdAttributeNode() - Объявляет заданный узал атрибута ключевым
  • DOMElement::setIdAttributeNS() - Объявляет атрибут с заданным локальным именем и URI пространства имен идентифицирующим
