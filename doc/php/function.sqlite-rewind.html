sqlite_rewind

SQLiteResult->rewind

(PHP 5, PECL sqlite >= 1.0.0)

sqlite_rewind -- SQLiteResult->rewind — Переходит к первой записи

Описание

bool sqlite_rewind ( resource $result )

Объектно-ориентированный стиль (метод):

bool SQLiteResult::rewind ( void )

sqlite_rewind() перемещает внутренний указатель в результирующем наборе на первую запись.

Список параметров

result

Ресурс результата SQLite. Этот параметр не требуется при использовании объектно-ориентированного метода.

Замечание:

Эта функция не может быть использована с идентификаторами результатов небуферизированных запросов.

Возвращаемые значения

Возвращает FALSE, если результат запроса не содержит записей, иначе возвращается TRUE.

Смотрите также

  • sqlite_next() - Переходит на следующую запись
  • sqlite_current() - Возвращает текущую запись из результирующего набора в виде массива
  • sqlite_seek() - Переходит к определенному номеру записи в буферизованном результирующем наборе
