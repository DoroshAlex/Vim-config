DomAttribute->specified

(PHP 4 >= 4.1.0)

DomAttribute->specified — Проверяет, задан ли атрибут

Описание

bool DomAttribute::specified ( void )

Проверяет, присвоено ли значение значение атрибуту в исходном документе.

Замечание:

Этот метод еще не реализован.

Возвращаемые значения

Возвращает TRUE в случае успешного завершения или FALSE в случае возникновения ошибки.
