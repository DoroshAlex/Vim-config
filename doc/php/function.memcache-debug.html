memcache_debug

(PECL memcache >= 0.2.0)

memcache_debug — Включить/выключить вывод отладочной информации

Описание

bool memcache_debug ( bool $on_off )

memcache_debug() включает вывод отладочной информации, если on_off установлен в TRUE и выключает, если в FALSE.

Замечание:

Функция memcache_debug() доступна, только если PHP был собран с использованием опции --enable-debug и в этом случае всегда возвращает TRUE. Иначе вызов этой функции ничего не делает и возвращает FALSE.

Список параметров

on_off

Включает вывод отладочной информации если уставновлено в TRUE. Выключает вывод отладочной информации если уставновлено в FALSE.

Возвращаемые значения

Возвращает TRUE, если PHP был собран с опцией --enable-debug, иначе - FALSE.
