DomAttribute->set_value

(PHP 4 >= 4.1.0)

DomAttribute->set_value — Устанавливает значение атрибута

Описание

bool DomAttribute::set_value ( string $content )

Эта функция устанавливает значение атрибута.

Список параметров

content

Новое значение.

Возвращаемые значения

Возвращает TRUE в случае успешного завершения или FALSE в случае возникновения ошибки.

Миграция на PHP 5

Установите значение свойства value класса DOMAttr.
