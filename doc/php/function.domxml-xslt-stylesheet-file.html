domxml_xslt_stylesheet_file

(PHP 4 >= 4.2.0)

domxml_xslt_stylesheet_file — Создает объект DomXsltStylesheet из XML файла

Описание

DomXsltStylesheet domxml_xslt_stylesheet_file ( string $xsl_file )

Создает объект класса DomXsltStylesheet из заданного XML файла.

Список параметров

xsl_file

Путь к XML файлу в виде строки.

Возвращаемые значения

Возвращает новый экземпляр класса DomXsltStylesheet.

Переход на PHP 5

Вызывайте XSLTProcessor::importStylesheet() с аргументом DOMDocument::load($xsl_file).
