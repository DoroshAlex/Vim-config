Настройка во время выполнения

Поведение этих функций зависит от установок в php.ini.

Параметры конфигурации Tidy
Название По умолчанию Изменяемость Журнал изменений
tidy.default_config "" PHP_INI_SYSTEM Доступен начиная с PHP 5.0.0.
tidy.clean_output "0" PHP_INI_USER PHP_INI_PERDIR в PHP 5. Доступен начиная с PHP 5.0.0.
Для подробного описания констант PHP_INI_*, обратитесь к разделу Где могут быть установлены параметры конфигурации.

Краткое разъяснение конфигурационных директив.

tidy.default_config string

Путь по умолчанию к конфигурационному файлу Tidy.

tidy.clean_output boolean

Включает/выключает востановление потока вывода средствами Tidy.

Внимание

Не следует включать tidy.clean_output в случае генерании контента, отличного от html, такого как динамические изображения.
