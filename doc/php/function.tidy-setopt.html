tidy_setopt

(PECL tidy >= 0.5.2)

tidy_setopt — Обновляет конфигурационные настройки для указанного документа tidy

Описание

bool tidy_setopt ( string $option , mixed $value )

Функция tidy_setopt() обновляет указанную опцию option новым значением value.

Список параметров

option

Название опции tidy. Список доступных конфигурационных опций может быть найден здесь: » http://tidy.sourceforge.net/docs/quickref.html.

value

Название опции tidy.

Возвращаемые значения

Возвращает TRUE в случае успешного завершения или FALSE в случае возникновения ошибки.

Примеры

Пример #1 Пример использования функции tidy_setopt()

<?php
$html = '<p>тест</i>';

$tidy = tidy_parse_string($html);

tidy_setopt('indent', FALSE);
?>

Примечания

Замечание: Эта функция доступна только в Tidy 1.0. С приходом Tidy 2.0 она стала ненужной и была удалена.
