Предопределенные константы

Перечисленные ниже константы определены данным расширением и могут быть доступны только в том случае, если PHP был собран с поддержкой этого расширения или же в том случае, если данное расширение было подгружено во время выполнения.

XSLT_OPT_SILENT (integer)
Отключение всего протоколирования и сообщений об ошибках. Эта настройка относится ко всем серверам (backends) и, возможно, будет добавлена в будущем.
XSLT_SABOPT_PARSE_PUBLIC_ENTITIES (integer)

Дать указание Sablotron разбирать все общедоступные сущности. По умолчанию этого не делается.

XSLT_SABOPT_DISABLE_ADDING_META (integer)

Не добавлять мета-тэг "Content-Type" в HTML вывод. Умолчания для этой настройки задаются во время компиляции библиотеки Sablotron.

XSLT_SABOPT_DISABLE_STRIPPING (integer)
Запретить удаление пробелов (только для файлов с данными).
XSLT_SABOPT_IGNORE_DOC_NOT_FOUND (integer)
Не считать фатальными неразобранные документы (функция document()).
XSLT_SABOPT_FILES_TO_HANDLER (integer)
XSLT_ERR_UNSUPPORTED_SCHEME (integer)
Код возврата при ошибке обработчиков схемы.
