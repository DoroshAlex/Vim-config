printer_abort

(PECL printer SVN)

printer_abort — Удаляет файл из буфера подготовки к печати

Описание

void printer_abort ( resource $printer_handle )

Эта функция удаляет файл из буфера подготовки к печати.

Список параметров

printer_handle

printer_handle должен быть действующим дескриптором принтера.

Возвращаемые значения

Эта функция не возвращает значения после выполнения.

Примеры

Пример #1 Пример printer_abort()

<?php
$handle = printer_open();
printer_abort($handle);
printer_close($handle);
?>
