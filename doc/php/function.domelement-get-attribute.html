DomElement->get_attribute

(PHP 4 >= 4.1.0)

DomElement->get_attribute — Возвращает значение заданного атрибута

Описание

string DomElement::get_attribute ( string $name )

Возвращает значение заданного атрибута в текущем элементе.

Начиная с PHP 4.3, если атрибутов с именем name не найдено, будет возвращена пустая строка.

Список параметров

name

Имя искомого атрибута. Этот аргумент чувствителен к регистру.

Возвращаемые значения

Возвращает значение атрибута в виде строки или пустую строку, если атрибутов с именем name не найдено.

Примеры

Пример #1 Получение значения атрибута

<?php

include("example.inc");

if (!$dom = domxml_open_mem($xmlstr)) {
    echo "Ошибка при разборе документа\n";
    exit;
}

// get chapter
$root = $dom->document_element();
echo $root->get_attribute('language'); // en

?>

Переход на PHP 5

Используйте DOMElement::getAttribute.
