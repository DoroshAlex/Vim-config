Предопределенные константы

Перечисленные ниже константы определены данным расширением и могут быть доступны только в том случае, если PHP был собран с поддержкой этого расширения или же в том случае, если данное расширение было подгружено во время выполнения.

RUNKIT_IMPORT_FUNCTIONS (integer)
Флаг runkit_import(), указывающий что необходимо импортировать функции из указанного файла
RUNKIT_IMPORT_CLASS_METHODS (integer)
Флаг runkit_import(), указывающий что необходимо импортировать методы классов из указанного файла
RUNKIT_IMPORT_CLASS_CONSTS (integer)
Флаг runkit_import(), указывающий что необходимо импортировать локальные константы классов из указанного файла. Эта возможность доступна начиная с версии PHP 5.1.0
RUNKIT_IMPORT_CLASS_PROPS (integer)
Флаг runkit_import(), указывающий что необходимо импортировать свойства классов из указанного файла.
RUNKIT_IMPORT_CLASSES (integer)
Флаг runkit_import(), представляющий побитовое ИЛИ для констант RUNKIT_IMPORT_CLASS_*.
RUNKIT_IMPORT_OVERRIDE (integer)
Флаг runkit_import(), указывающий что необходимо перезаписывать существующие функции, методы, константы и свойства классов. Если этот флаг не задан, перезапись существующих определений производиться не будет.
RUNKIT_ACC_PUBLIC (integer)
Флаг runkit_method_add(), специфичный для PHP 5
RUNKIT_ACC_PROTECTED (integer)
Флаг runkit_method_add(), специфичный для PHP 5
RUNKIT_ACC_PRIVATE (integer)
Флаг runkit_method_add(), специфичный для PHP 5
CLASSKIT_ACC_PUBLIC (integer)
Флаг runkit_method_add(), специфичный для PHP 5. Определен только при включенной совместимости с classkit.
CLASSKIT_ACC_PROTECTED (integer)
Флаг runkit_method_add(), специфичный для PHP 5. Определен только при включенной совместимости с classkit.
CLASSKIT_ACC_PRIVATE (integer)
Флаг runkit_method_add(), специфичный для PHP 5. Определен только при включенной совместимости с classkit.
CLASSKIT_AGGREGATE_OVERRIDE (integer)
Флаг classkit_import(), специфичный для PHP 5. Определен только при включенной совместимости с classkit.
RUNKIT_VERSION (string)
Представляет текущую версию runkit.
CLASSKIT_VERSION (string)
Представляет текущую версию runkit. Определен только при включенной совместимости с classkit.
