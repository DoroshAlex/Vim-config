sqlite_has_prev

SQLiteResult->hasPrev

(PHP 5)

sqlite_has_prev -- SQLiteResult->hasPrev — Определяет, доступна ли предыдущая запись

Описание

bool sqlite_has_prev ( resource $result )

Объектно-ориентированный стиль (метод):

bool SQLiteResult::hasPrev ( void )

Определяет, доступны ли предыдущие записи в указанном результате.

Список параметров

result

Ресурс результата SQLite. Этот параметр не требуется при использовании объектно-ориентированного метода.

Замечание:

Эта функция не может быть использована с идентификаторами результатов небуферизированных запросов.

Возвращаемые значения

Возвращает TRUE если предыдущая запись результата запроса result доступна, иначе возвращает FALSE.

Смотрите также

  • sqlite_prev() - Переходит на предыдущую запись
  • sqlite_has_more() - Определяет, есть ли еще записи в результате запроса
  • sqlite_num_rows() - Возвращает количество записей в результате запроса
