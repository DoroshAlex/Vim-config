mysqli_get_metadata

(PHP 5)

mysqli_get_metadata — Синоним для mysqli_stmt_result_metadata()

Описание

Функция является синонимом mysqli_stmt_result_metadata().

Примечания

Замечание:

Функция mysqli_get_metadata() устарела и будет удалена.
