Imagick::blackThresholdImage

(PECL imagick 2.0.0)

Imagick::blackThresholdImage — Перевести все пиксели ниже порогового значения в черный цвет

Описание

bool Imagick::blackThresholdImage ( mixed $threshold )

Работает так же, как Imagick::thresholdImage(), но изменяются только пиксели ниже порогового значения, в то время как остальные пиксели остаются неизменными.

Список параметров

threshold

Пороговое значение ниже которого все пиксели станут черными

Возвращаемые значения

В случае успешной работы возвращает TRUE.

Список изменений

Версия Описание
2.1.0 В качестве параметра возможно передавать цвет строкой. В предыдущих версиях разрешалось передавать только ImagickPixel.
