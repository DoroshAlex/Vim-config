dngettext

(PHP 4 >= 4.2.0, PHP 5)

dngettext — Версия dgettext для множественного числа

Описание

string dngettext ( string $domain , string $msgid1 , string $msgid2 , int $n )

Функция dngettext() позволяет переопределить текущий домен domain для одного сообщения во множественном числе.

Список параметров

domain

Домен.

msgid1

msgid2

n

Возвращаемые значения

В случае успеха возвращает значение типа string.
