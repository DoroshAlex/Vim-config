Установка

Для установки расширения AMQP для PHP вам необходимо сначала установить библиотеку » librabbitmq. Используйте следующие шаги, чтобы загрузить и установить библиотеку: 

# Загрузка библиотеки rabbitmq-c
  hg clone http://hg.rabbitmq.com/rabbitmq-c/rev/3c549bb09c16 rabbitmq-c
  cd rabbitmq-c
  # Добавить требование codegen
  hg clone http://hg.rabbitmq.com/rabbitmq-codegen/rev/f8b34141e6cb codegen
  # Конфигурирование, сборка и установка
  autoreconf -i && ./configure && make && sudo make install

Информация по установке этого расширения PECL может быть найдена в главе руководства Установка PECL расширений. Дополнительная информация, такая как новые версии, скачивание, исходные файлы, информация о разработчике и CHANGELOG, может быть найдена здесь: » http://pecl.php.net/package/amqp.

Примечание для пользователей Windows: Это расширение в настоящий момент не поддерживается в Windows, так как библиотека librabbitmq ещё не поддерживается в Windows.
