hexdec

(PHP 4, PHP 5)

hexdec — Переводит число из шестнадцатиричной системы счисления в десятичную

Описание

number hexdec ( string $hex_string )

Возвращает десятичный эквивалент шестнадцатеричного числа, содержащегося в аргументе hex_string. hexdec() преобразует шестнадцатеричную строку в десятичное число.

hexdec() игнорирует любые обнаруженные не-шестнадцатеричные символы.

Список параметров

hex_string

Шестнадцатеричная строка для преобразования

Возвращаемые значения

Десятичное представление hex_string

Список изменений

Версия Описание
С 4.1.0 Теперь функция может преобразовывать числа, слишком большие для того, чтобы вместиться в тип integer для имеющейся платформы. В этом случае значения возвращаются как тип float.

Примеры

Пример #1 Пример использования hexdec()

<?php
var_dump(hexdec("See"));
var_dump(hexdec("ee"));
// в обоих случаех будет выведено "int(238)"

var_dump(hexdec("that")); // выведет "int(10)"
var_dump(hexdec("a0")); // выведет "int(160)"
?>
