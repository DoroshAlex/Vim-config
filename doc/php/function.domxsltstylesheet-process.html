DomXsltStylesheet->process

(PHP 4 >= 4.1.0)

DomXsltStylesheet->process — Применяет XSLT-преобразование к объекту DomDocument

Описание

DomDocument DomXsltStylesheet::process ( DomDocument $xml_doc [, array $xslt_params [, bool $is_xpath_param [, string $profile_filename ]]] )

Применяет XSLT-преобразование к заданному объекту класса DomDocument.

Список параметров

xml_doc

XML документ для преобразования в виде объекта класса DomDocument.

xslt_params

Ассоциативный массив содержащий имена и значения параметров.

is_xpath_param

Если установлен в FALSE, значения массива xslt_params будут заключаться в кавычки. Это поведение по умолчанию. Оно позволяет передать значения в виде PHP строк.

Замечание:

Если строки содержат одновременно и одинарные, и двойные кавычки, необходимо самостоятельно заключить все значения в кавычки и установить значение этого аргумента в TRUE.

profile_filename

Задайте путь к файлу, если требуется сведения профилирования.

Возвращаемые значения

Возвращает результат обработки в виде объекта класса DomDocument.

Переход на PHP 5

Используйте XSLTProcessor::setParameter() и XSLTProcessor::transform-to-doc().

Список изменений

Версия Описание
4.3.0 Добавлен аргумент profile_filename.
