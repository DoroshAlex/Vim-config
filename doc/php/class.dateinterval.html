Класс DateInterval

(PHP 5 >= 5.3.0)

Введение

Представление интервалов дат. Интервалы дат хранятся либо определенном фиксированном времени (в годах, месяцах, днях, часах и т.д.) или относительной строке времени, формат который поддерживает DateTime конструктор.

Обзор классов

DateInterval {
/* Свойства */

public integer $h ;

public mixed $days ;

}

Свойства

y

Количество лет.

m

Количество месяцев.

d

Количество дней.

h

Количество часов.

i

Количество минут.

s

Количество секунд.

invert

Принимает 1 если интервал инвертируется и 0 в противном случае. См. DateInterval::format().

days

Суммарное количество дней между начальной и конечной датами в DateTime::diff() вычислении. В других случаях days будет FALSE.

Содержание

  • DateInterval::__construct — Создает новый объект DateInterval
  • DateInterval::createFromDateString — Создает объект класса DateInterval, используя данные из переданной строки
  • DateInterval::format — Форматирует интервал
