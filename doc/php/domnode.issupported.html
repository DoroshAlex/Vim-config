DOMNode::isSupported

(PHP 5)

DOMNode::isSupported — Проверяет, поддерживается ли заданное свойство в определенной версии

Описание

bool DOMNode::isSupported ( string $feature , string $version )

Проверяет, поддерживается ли заданное свойство feature в определенной версии version.

Список параметров

feature

Название свойства для проверки. Список названий модулей приведен в примере к DOMImplementation::hasFeature().

version

Номер версии модуля (feature) для проверки.

Возвращаемые значения

Возвращает TRUE в случае успешного завершения или FALSE в случае возникновения ошибки.
