OCI-Lob->erase

(PHP 5, PECL OCI8 >= 1.1.0)

OCI-Lob->erase — Очищает указанную часть объекта LOB

Описание

int OCI-Lob::erase ([ int $offset [, int $length ]] )

Очищает конкретную часть объекта LOB, начиная с позиции, определенной в параметре offset. По умолчанию LOB очищается полностью.

Для объектов типа BLOB очищение означает заполнение нулевыми байтами. Объекты типа CLOB заполняются пробелами.

Список параметров

offset

length

Возвращаемые значения

Возвращает число очищенных символов/байт или FALSE в случае возникновения ошибки.
